# 내 손안의 개인 트레이너: MyPT

## 프로젝트 소개
내 손안의 개인 트레이너, MyPY 앱은 인공지능 트레이너 모델을 이용해, 장병들의 운동 자세를 분석하고, 피드백함으로써 운동하는 장병들의 부상을 방지하고, 순위보드를 제공하여 체력 단련을 고취시키는 앱입니다.
